This navigation is based on "Mobile Dropdown Conversion" by Astuteo and modified by Paul Cheney.

To view the navigation, make your browser narrow and try it out.

I have made the following changes to his code base on my Github account.

  • Changed the graphics used for the navigation
  • Removed the dependency on the jQuery library
  • Updated for natural box layout model
  • Pixel units replaced with rem units
  • Changed the parent down arrow to a hex code "\25BC"
  • Limited the dropdown width
  • Adapted for a flat design look
  • Added styling for an "active" state on the menu list item
